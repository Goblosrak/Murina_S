Лабораторная работа по изучению сериализации/десериализации JSON, управления ресурсами и работы с файловой системой в C#. Реализованы три основных класса:<br>
- "Person" - класс с данными пользователя с атрибутами JSON<br>
- "PersonSerializer" - менеджер сериализации с синхронными и асинхронными операциями<br>
- "FileResourceManager" - менеджер файловых ресурсов с паттерном IDisposable<br><br>

Технологии и библиотеки<br>
- Язык: C# 10.0<br>
- Платформа: .NET 10.0<br>
- Библиотеки:<br>
- "System.Text.Json" - для сериализации JSON<br>
- "System.IO" - для работы с файлами и потоками<br>
- "System.Threading.Tasks" - для асинхронных операций<br><br>
Инструкция по запуску<br>
- Нажать треугольник(Run), находясь в Program.cs
