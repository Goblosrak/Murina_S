Лабораторная работа 3: Реализация коллекций в C#
Выполнила: Мурина Софья К0709-23/3

Цель работы
    Познакомиться с интерфейсами коллекций в C# (.NET), реализовать простые коллекции и понять их контракт и функциональность.

Основные интерфейсы для изучения
    IEnumerable — поддержка перебора элементов (foreach)
    ICollection — базовые операции: добавление, удаление, подсчёт элементов
    IList — индексированный доступ, вставка, удаление по индексу
    IDictionary<TKey, TValue> — словарь, работа с ключами и значениями

Задание
    1. Реализовать класс SimpleList
        Реализует интерфейсы IEnumerable, ICollection, IList
        Внутренне использует динамический массив (без готовых коллекций List и т.п.)
        Поддерживает:
        Добавление элементов
        Получение/установку по индексу
        Удаление элементов
        Поиск индекса элемента
        Реализовать методы и свойства: Count, Add, Remove, Clear, Contains, IndexOf

    2. Реализовать класс SimpleDictionary<TKey, TValue>
        Реализует IDictionary<TKey, TValue> (собственный массив с цепочками)
        Поддерживает добавление, удаление и поиск по ключу
        Обеспечить перебор (IEnumerable<KeyValuePair<TKey, TValue>>)
        Реализовать поддержку IReadOnlyCollection / IReadOnlyList
        Реализовать свою версию двунаправленного связного списка с интерфейсом IList

    3. Требования к тестам
        Написать автоматизированные тесты, которые проверяют корректность всех операций
        Тесты должны быть детерминированными и воспроизводимыми (одинаковые входные данные → одинаковые ожидаемые результаты)
        Использовать фреймворки для тестирования (Я выбрала xUnit)

Технологии
    .NET 6.0+ / .NET Core 3.1+
    xUnit
    C# 9.0+

Инструкция по запуску тестов
